# M+F Property Management Website

## Project Documentation

The M+F Property Management website provides a user-friendly platform for managing properties, tenants, and maintenance requests. The site includes features for landlords and tenants to communicate effectively and manage their accounts.

### Key Features:
- User authentication and authorization
- Property listing and management
- Tenant profile management
- Maintenance request tracking
- Secure payment processing

## Setup Instructions

To set up the M+F Property Management website locally, follow these instructions:

1. **Clone the repository:**  
   ```bash
   git clone https://github.com/M-Fpm/M-F-Property-Management.git
   cd M-F-Property-Management
   ```

2. **Install dependencies:**  
   Make sure you have [Node.js](https://nodejs.org/) installed, then run:  
   ```bash
   npm install
   ```

3. **Set up environment variables:**  
   Create a `.env` file and configure the necessary environment variables based on the `.env.example` file provided in the repository.

4. **Run the development server:**  
   ```bash
   npm start
   ```
   Visit `http://localhost:3000` in your browser to see the application in action.

5. **Build for production:**  
   To create an optimized build for deployment, run:  
   ```bash
   npm run build
   ```
